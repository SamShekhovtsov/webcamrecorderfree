# Web camera recorder free
## Recording laptop webcam surveillance system

this software is free to use, 
it was created for travelers to save life during travels.

I have noticed that my kidney is damaged in the night during travels in Asia and Africa.
That is why I have decided to build a software that can record laptop camera video
when I go sleeping.
This software runs on laptop and records laptop web camera video to *.avi files.
There are options for the video quality, for the recorded video files storage 
and an option for the split files size (chunk size).
This is surveillance software which is built to save peoples' life.

## Recording all night 10+ hours
## Saving to split video files with very compact compression and adequate size
## Optional video quality / resolution and split files size

### supported video resolutions:
 > (SD) 480p (640x480) - very compact, minimal video file size but low video quality
 > (HD) 720p(1280x720) 
 > Full HD (FHD) 1080p (1920x1080)
 > Ultra HD (UHD) 4K (3840x2160) - high resolution videos. Large video files.
 > 8K (7680x4320) - highest supported resolution, very large size of the video files
